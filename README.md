# chat-aggregator

Каркас проекта агрегатора чатов (Telegram, WhatsApp Web, Bitrix24, MAX).

## Технологии
- Backend: FastAPI
- Frontend: React + TypeScript + PWA

## Структура
```text
chat-aggregator/
  backend/
  frontend/
  docs/
  infra/
  scripts/
  tests/
  storage/
  .env.example
  .gitignore
  docker-compose.yml
  README.md
```

## Запуск (базово)
Проект содержит только каркас без бизнес-логики.
